# Timephile Backend

This complements https://github.com/aloder/timephile-frontend

### Prerequisates
* Docker
* Docker-compose
* Node 10.4.1
* Yarn
* Prisma

### Installation
1. Navigate to the root directory
2. copy the .env.inc and fill the secrets
3. docker-compose up -d
4. prisma deploy
5. yarn start
